Lab Updation Software

Lab updation software is designed to provide lab technicians and managers with a simple, intuitive interface that allows them to easily view and update the status of orders.

Procedure

  1. To open the lab updation portal we will click on the below-given link.
  2. Click here

  3. After clicking on the above link this screen will appear.

  4. There is a checkbox(VRX AD User) on the right side of the website, Click on the VRX AD User checkbox.

  5. Now the screen will look like this

  6. Now enter your Login ID In the User Name field.

  7. After entering the Username, click on the Login Button.

  8. After clicking on the Login button, the Dashboard screen will appear like this.

  9. "In the left side of the screen in the sidebar, there is a Process button, click on this button.

  10. After clicking on the Process button, this screen will appear.

  11. Now click on the Update Domestic Site Status Button.

  12. After clicking on the "Update Domestic Site Status Button", this screen will appear.

  13. In this screen, There is a Dropdown menu to select the Lab, click on this dropdown menu button.

  14. After clicking on the dropdown menu button, there will be a list of options to select, these options will be different for the users of different departments.

    (For example if we login as Surfacing Lab user, the options will be shown only related to the Surfacing lab, similarly for other departments like Coating, Tinting etc.)

  15. Select the option as per the requirement, in this example, we are logged in as a Digital Surfacing department user, and we are issuing the order to QC DS Gurgaon, So we have selected Option "DG15-GF-01-Issued To QC-DRS-GRG.

  16. Now after selecting the Lab option, we will enter the OCI number of a particular order.

  17. In this screen, There is a Dropdown menu to select the Lab, click on this dropdown menu button.

  18. After entering the OCI Number, click on the Update Status button, by clicking this button status of particular order will be updated.

  19. After clicking the Update Status button, If the OCI number and the selected lab from dropdown menu are valid, the status of the order will be updated successfully. (Order Status: Success)

  20. If the OCI number is not valid, then this screen will appear, and the status of the order will be this: Failure - OCI Number is not valid.

  21. If we select the wrong Lab option from the dropdown menu for any OCI Number then the status of the order will be not updated.

    For Example: If the lens is currently in the invenotry and we want to update the status of this order to Issue to surfacing QC without sending it to surfacing lab, then this will throw error Failure: At inventory status is not valid state for Lab.

  22. If the order is in the Surfacing QC and we will directly update its status to - Dispatch to Branch for Fitting, this will throw this error: Coating/Tinting/Fitting Status Log Validation Error.
